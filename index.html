📌 Exercício 1

Enunciado: Armazene em um vetor os nomes de 5 alunos, liste todos em uma UL e mostre a quantidade de alunos cadastrados.

✅ Resposta:

Lista de Alunos:

    Total de alunos: 0

    // Código:
const alunos = ["Ana", "Bruno", "Carla", "Daniel", "Elisa"];
const lista = document.getElementById('listaAlunos');
for (let i = 0; i < alunos.length; i++) {
    let item = document.createElement('li');
    item.textContent = alunos[i];
    lista.appendChild(item);
}
document.getElementById('totalAlunos').textContent = alunos.length;

    📌 Exercício 2

    Enunciado: Crie um vetor contendo as notas [25, 80, 65, 90, 100]. Calcule e exiba a média.

    ✅ Resposta:

    Notas:

    Média:

    Cálculo: 25 + 80 + 65 + 90 + 100 = 360 ÷ 5 = 72

    // Código:
const notas = [25, 80, 65, 90, 100];
let soma = 0;
for (let i = 0; i < notas.length; i++) {
    soma += notas[i];
}
const media = soma / notas.length;
console.log("Média:", media); // 72

    📌 Exercício 3

    Enunciado: Crie um vetor com os produtos: ["Arroz", "Feijão", "Macarrão", "Leite", "Café"]. Permita que o usuário digite um produto e verifique se ele existe no vetor.

    ✅ Resposta:

    Produtos disponíveis: Arroz, Feijão, Macarrão, Leite, Café

    // Código:
const produtos = ["Arroz", "Feijão", "Macarrão", "Leite", "Café"];
function buscarProduto() {
    let digitado = document.getElementById('produtoInput').value.trim();
    let encontrado = false;
    for (let i = 0; i < produtos.length; i++) {
        if (produtos[i].toLowerCase() === digitado.toLowerCase()) {
            encontrado = true;
            break;
        }
    }
    alert(encontrado ? "Produto encontrado!" : "Produto não encontrado!");
}

    📌 Exercício 4

    Enunciado: Crie um programa que receba o vetor [10, 5, 8, 10, 3, 5, 7, 8]. O programa deve: criar um novo vetor sem valores repetidos, exibir o vetor original, o vetor sem duplicatas e a quantidade de elementos removidos.

    ✅ Resposta:

    Vetor original: (8 elementos)

    Vetor sem duplicatas: (5 elementos)

    Elementos removidos: 

    // Código:
const original = [10, 5, 8, 10, 3, 5, 7, 8];
const semDuplicatas = [];
for (let i = 0; i < original.length; i++) {
    let jaExiste = false;
    for (let j = 0; j < semDuplicatas.length; j++) {
        if (semDuplicatas[j] === original[i]) {
            jaExiste = true;
            break;
        }
    }
    if (!jaExiste) semDuplicatas.push(original[i]);
}
const removidos = original.length - semDuplicatas.length;
console.log("Removidos:", removidos); // 3
